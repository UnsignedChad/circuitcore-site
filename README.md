# circuitcore-site

Landing page for [circuitcore](https://github.com/UnsignedChad/circuitcore).

Plain HTML + one CSS file, hosted on GitHub Pages. Layout follows
notepad-plus-plus.org (top nav, left sidebar, news-style content
column) but dark themed.

## Local preview

```
python3 -m http.server 8000
```

then open <http://localhost:8000>.

## Deploy

GitHub Pages serves whatever sits on the `main` branch. Push a commit,
wait ~30 seconds, refresh the live site.
