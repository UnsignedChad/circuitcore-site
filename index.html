What is it

circuitcore is an open-source PCB analysis suite that consumes a KiCad .kicad_pcb directly and runs the kind of signal-integrity, power-integrity, EMI and multiphysics analyses you would otherwise pay six figures for. Five tabs in one Qt6 desktop shell, one shared board model, headless CLI for each kit, GPL-3.0 throughout.

The design philosophy: trust the board file, skip the schematic ceremony, ship analytical solvers where the math is closed-form, and reach for FEM / FDTD only where the physics actually demands it.

The kits

sikit Signal integrity. Trace impedance (Wadell + FDM), differential pairs, stat-eye, crosstalk, Touchstone + IBIS-AMI, return-path checks, FDTD3D.
pdnkit Power integrity. IR-drop FVM, cavity Z(f), decoupling-cap optimiser, plane resonances, MOR, transient PDN.
emikit EMI pre-compliance. Loop emissions, common-mode cable model, CISPR / FCC masks, ground-bounce envelope.
mpkit Multiphysics. Voxel hex mesh, steady-state and transient heat, linear elasticity, thermoelectric post-processor, IR-drop → Joule coupling.
studio Unified Qt6 GUI. Five tabs (Board / SI / PI / EMI / Mp), one shared BoardModel, drag-and-drop a .kicad_pcb to start.

Download

No prebuilt binaries yet. Clone from source while the Windows port settles: 

git clone https://github.com/UnsignedChad/circuitcore.git
cd circuitcore
cmake -B build -G Ninja
cmake --build build -j 8
ctest --test-dir build

See the build section below for platform-specific dependencies.

Build from source

Linux (Debian / Ubuntu)

sudo apt install qt6-base-dev qt6-base-dev-tools \
    libqt6opengl6-dev libeigen3-dev libspdlog-dev \
    libcli11-dev catch2 libsuitesparse-dev \
    cmake ninja-build g++-13

Windows (Visual Studio + vcpkg)

Open the repo as a CMake project in Visual Studio 2022 or later. With the vcpkg toolchain wired in (Project → CMake Settings → CMake toolchain file pointing at C:/vcpkg/scripts/buildsystems/vcpkg.cmake), the vcpkg.json manifest in the repo root auto-installs Qt6, Eigen, spdlog, CLI11 and Catch2 on the first configure. Enable Windows long-path support before building or the VTK fetch will hit MAX_PATH.
